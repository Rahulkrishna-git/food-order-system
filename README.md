# Food Order System

## Project Description
This is a Java Swing + MySQL based Food Order Management System.

It performs:
- Insert food orders
- Update food orders
- Delete food orders
- View all orders in JTable

## Technologies Used
- Java
- Swing
- JDBC
- MySQL
- VS Code

## Database Table
Table Name: food_orders

Columns:
- restaurant_name
- food_item
- quantity
- total_price
- address

## Features
- GUI using Java Swing
- MySQL database connectivity
- CRUD Operations
- Auto refresh after update

## How to Run
Compile:
```bash
javac -cp ".;mysql-connector-j-8.x.x.jar" FoodOrdersSwing.java
